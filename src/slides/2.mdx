## Le sha1

Il permet d'identifier les éléments git
* peu de colisions
* chaque contenu son sha1
* chaque sha1 son contenu
* les objets git sont immuables

---

## Le blob

* l'objet de base Git
* contient du contenu text
* calcule le sha1 du contenu

```sh
git cat-file -p <sha1>
```
